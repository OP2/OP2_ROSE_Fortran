Bugs in Current Release of POP

The following are bugs that have been reported and not yet fixed in the current release of POP. If you have encountered a bug which does not appear on this list, please send e-mail to climate@acl.lanl.gov with a detailed description of the bug together with the machine and model configuration you were running.

  • restart failure at small subgrids
  • barotropic failure with zero forcing

Details of each bug

Title: restart failure at small subgrids

  • Submitted: 1999/11/12
  • Description: restart I/O causes the code to fail with KE > 100 when running problems with small subgrids (buffers being overwritten? WORKAROUND: the problem disappears with num_iotasks=1
    • . Use of the PSF library will also eliminate this problem and is recommended on machines where this library is available.
